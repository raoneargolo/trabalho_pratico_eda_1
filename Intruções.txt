Componentes : Acácia Juriti , Eric Sampaio, Raone Argolo

Uso do programa:
1- Para compilar o aquivo basta dar o seguinte comando:  gcc –x c “nomedearquivo.c” –o “nomedearquivo.exe”.
2- Depois de aberto, o programa pedirá o nome do aquivo, basta dizer o nome do arquivo. extensao Ex.: (aquivo.csv ou aquivo.txt)
3- Escolha uma das estruturas para manipular ( Árvores ou Listas Duplamente encadeada)
4- Aparecerá um menu com as opções disponiveis: (1-Buscar por Matricula, 2-Buscar por nome,3-imprimir, 4-...)
5- A cada opcao o programa executará o que foi solicitado o depois perguntará o que deseja fazer novamente
6-Para sair do programa será feito uma volta de cada submenu, sendo assim no menu principal escolher a opção 6 ( sair)
